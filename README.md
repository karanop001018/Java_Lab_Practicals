## Getting Started

1. **Clone the Repository:**
   ```
   git clone https://github.com/karanop001018/Java_Lab_Practicals.git
   ```

2. **Navigate to the Practical You Want:**
   ```
   cd Java_Lab_Practicals/practical_1
   ```

   ```
   javac practical_1.java
   java practical_1
   ```
